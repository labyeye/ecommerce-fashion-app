# Koolegant Font Files

Please add your Koolegant font files to this folder with the following naming convention:

## Required Font Files:

### WOFF2 Format (Best compression, modern browsers):
- `Koolegant-Thin.woff2` (font-weight: 100)
- `Koolegant-ExtraLight.woff2` (font-weight: 200)
- `Koolegant-Light.woff2` (font-weight: 300)
- `Koolegant-Regular.woff2` (font-weight: 400)
- `Koolegant-Medium.woff2` (font-weight: 500)
- `Koolegant-SemiBold.woff2` (font-weight: 600)
- `Koolegant-Bold.woff2` (font-weight: 700)
- `Koolegant-ExtraBold.woff2` (font-weight: 800)
- `Koolegant-Black.woff2` (font-weight: 900)

### WOFF Format (Fallback for older browsers):
- `Koolegant-Thin.woff`
- `Koolegant-ExtraLight.woff`
- `Koolegant-Light.woff`
- `Koolegant-Regular.woff`
- `Koolegant-Medium.woff`
- `Koolegant-SemiBold.woff`
- `Koolegant-Bold.woff`
- `Koolegant-ExtraBold.woff`
- `Koolegant-Black.woff`

### TTF Format (Final fallback):
- `Koolegant-Thin.ttf`
- `Koolegant-ExtraLight.ttf`
- `Koolegant-Light.ttf`
- `Koolegant-Regular.ttf`
- `Koolegant-Medium.ttf`
- `Koolegant-SemiBold.ttf`
- `Koolegant-Bold.ttf`
- `Koolegant-ExtraBold.ttf`
- `Koolegant-Black.ttf`

## Minimum Required Files:
If you don't have all weights, at least add these essential ones:
- `Koolegant-Regular.woff2` (400)
- `Koolegant-Medium.woff2` (500)
- `Koolegant-Bold.woff2` (700)

## After Adding Font Files:
1. Restart your development server
2. The Koolegant font will be automatically applied to your entire website
3. No additional code changes needed!

## Font Conversion:
If you only have .otf or other formats, you can convert them to web formats using:
- [Convertio](https://convertio.co/otf-woff2/)
- [CloudConvert](https://cloudconvert.com/otf-to-woff2)
- [FontSquirrel Webfont Generator](https://www.fontsquirrel.com/tools/webfont-generator)
